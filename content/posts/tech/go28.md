# 总结
- make和new都是golang用来分配内存的內建函数，<font color="red">且在堆上分配内存，make 即分配内存，也初始化内存。new只是将内存清零，并没有初始化内存。</font>
- make返回的还是引用类型本身；而**new返回的是指向类型的指针。指针值为0xc00004c088，这个指针指向的内容的值为零（zero value）**, 零值 （*ptr）就是对应类型各个初始化值
- make只能用来分配及初始化类型为slice，map，channel的数据；new可以分配任意类型的数据。

**go会给引用类型初始化为nil，nil是不能直接赋值的。并且不能用new分配内存 (指向类型的指针),只能使用make分配引用类型的内存, 无法直接赋值。(与值类型区分)**


# new 函数
new(T)函数是一个分配内存的内建函数。指针变量用来接收

## 1.26版本 针对new进行优化

- 我们都知道，对于一个已经存在变量，可对其指针进行赋值？ 可以
``` go
var p int 
var v *int
v = &p // tag: 已经存在的变量 指向int类型的指针。赋值的时候取已经存在的int变量的地址赋值
*v = 11
fmt.Println(*v) // 11
```

- 不是已经存在的变量会如何呢？能对其直接赋值吗？-不能
``` go
var v *int
*v=8
fmt.Println(*v) // panic: runtime error: invalid memory address or nil pointer dereference
// [signal 0xc0000005 code=0x1 addr=0x0 pc=0x48df66]
```

- 优化写法：通过new来初始化一地址，再赋值

## new 普通类型
- 初始化一个指针变量，其值为nil，nil的值是不能直接赋值的。通过new其返回一个指向新分配的类型为int的指针，这个指针指向的内容的值为零（zero value）

``` go
func TestNew(t *testing.T) {
	var a *int
	// fmt.Println(*a) // tag: 此时还没分配内存 panic: runtime error: invalid memory address or nil pointer dereference
	fmt.Printf("%v, %#v\n", a, a) // <nil>, (*int)(nil)

	b := new(int)                   // 指向类型的指针 分配内存并清零
	fmt.Printf("%v, %#v\n", b, b)   // 0x1400000e258, (*int)(0x1400000e258)
	fmt.Printf("%v, %#v\n", *b, *b) // 0, 0
}
```
- 不同的指针类型零值是不同的
``` go
type Name struct {
    P string
}
var av *[5]int 
var iv *int 
var sv *string 
var tv *Name

av = new([5]int)
fmt.Println(*av) // [0 0 0 0 0 0]
iv = new(int)
fmt.Println(*iv) // 0
Sv = new(string)
fmt.Println(*sv) //
tv = new(Name)
fmt.Println(*tv) // {}
```

## new 复合类型（数组，引用类型slice，map，channel等）
- 声明后再内存清零
- 数组通过new处理，数组av初始化零值，数组虽然是复合类型，但不是引用类型，所以数组可以通过指向地址赋值或者访问
- 其他silce、map、channel类型也属于引用类型，go会给引用类型初始化为nil，nil是不能直接赋值的。并且<font color="red">不能用new分配内存, 无法通过指向变量的指针进行直接赋值</font>

## <font color="red">*配合new使用</font>
- 注意之前直接new切片 map后赋值报错的对比, (*mv) 是mv指针类型指向的变量
``` go
var mv *map[string]string // 指针类型变量
fmt.Printf("mv: %p %#v \n". &mv， mv)//mv: Oxc042004028 (*map[string]string)(nil) 

mv = new(map[string]string)
fmt. Printf("mv: %p %#v n"， &mv， mv)//mv: Oxc000006028 &map[string]string(nil) 

// 通过make重新进行内存分配
(*mv) = make(map[string]string) // (*mv) 是mv指针类型指向的变量
(*mv)["a"] = "a"
fmt. Printf("mv: %p %#v \n" &mv, mv)//mv: Oxc042004028 &map[string]string{"a" :"a"}
```
